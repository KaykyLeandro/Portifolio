# Portifolio Front-end
## Kayky Leandro

- Esse é um portifolio front-end, responsivo, feito usando **Next.JS** (e um pouco da API).

Trata-se de uma SPA (Single Page Application - Aplicação de uma única página), está usando o biblioteca **React** e o framework **Next.JS** por causa das suas funcionalidades.
O projeto foi feito usando **typescript**, e tailwind para estilização.

- Esse é meu primeiro portifólio... e segue em constante evolução até ser hospedado. Se quiser, sinta-se a vontade para comentar, e sugerir mudanças ou melhorias ^_^
